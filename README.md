# Online Study Planner

A modern web-based platform for students to organize subjects, track tasks, manage timetables, and achieve academic goals.

## Quick start

1. Clone the repository to your local machine.
2. Open `html/main.html` in your web browser.
3. Navigate through the sidebar to access different features.

## Project layout

- `html/`: HTML pages and templates
- `css/`: Stylesheets for layout and design
- `js/`: JavaScript files for validation and functionality
- `public/`: Images and static assets
- `xml/`: XML data files and DTD schemas

## Features

- Real-time form validation with visual feedback
- Smart search functionality across subjects, tasks, and goals
- Responsive design with modern UI components
- Weekly study planner with XML data structure
- Subject management with educational resource links
- User authentication with secure password requirements

## Technical Stack

- HTML5 with frame-based architecture
- CSS3 with gradients, glassmorphism, and animations
- Vanilla JavaScript for client-side validation
- XML/DTD for structured data storage

## Notes

- Form validation includes email format and password complexity checks.
- Search feature filters across multiple content types in real-time.
- Responsive design adapts to different screen sizes automatically.
